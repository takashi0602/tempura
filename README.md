```

```

```

```

```

```

```

```

```
npm install gulp-pug
```

```
npm install browser-sync
```

# Gulpコマンド

ローカルサーバの起動

```
gulp
```

PugとSassの自動コンパイル

```
gulp watch
```

PugとSassのコンパイル

```
gulp compile
```
